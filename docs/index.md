## Overview

CaveGenLabs (CGL) provides a consistent environment for implementing, visualizing, comparing, and exploring cave generation algorithms without the overhead of building a separate application for each algorithm. 

---

## Why CaveGenLabs?

CaveGenLabs was designed around a simple idea: experimenting with procedural cave generation should be easy.

Instead of building a new application for every algorithm, CGL provides a shared environment with a common architecture, allowing algorithms to be implemented, tested, and compared.

Key features include:

- A modular architecture for implementing new algorithms.
- A common parameter and configuration system.
- Interactive visualization for generated cave layouts.
- An open-source codebase designed for experimentation and learning.

---

## Included Algorithms

CGL currently includes the following procedural cave generation algorithms:

- *Random Walk*
- *Midpoint Displacement*
- *Cellular Automata*
- *Perlin Worms*
- *Binary Space Partitioning*
- *Metaballs*
- *Diffusion Limited Aggregation*

See the [Algorithms](algorithms.md) section for a detailed overview of each implementation.
